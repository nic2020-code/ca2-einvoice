Hướng dẫn hồ sơ

Để phát hành hóa đơn điện tử bạn có thể tải toàn bộ - trong đó bao gồm:
1 - Quyết định sử dụng - File XML / hoặc bản in và đóng dấu / hoặc cả 2 - Thực hiện tại bước 1
2- Thông báo phát hành - File XML / hoặc bản in và đóng dấu / hoặc cả 2 - Thực hiện tại bước 2
3- Mẫu hóa đơn điện tử - File XML / hoặc bản / hoặc cả 2 - Thực hiện tại bước 3
